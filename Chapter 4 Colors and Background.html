We can also use 'background-repeat' property in CSS to repeat a background in a x/y (horizontal/vertical) direction. Just use 'body' selector in CSS and put that any image as the background.


There is another property called 'background-size' property which takes cover/contain/auto/width as value.

  • cover = fits and no empty space remains, without destroying the aspect ratio
  • contain = fits and image is fully visible
  • auto = Display in original size
  • width = Set width of the image and the height will be automatically set according to the aspect ratio

There is a property called border which initiates a border aroung the background color box. Just like in this. It is written as 'border : border_weight border_type border_color'. The border_weight defines the thickness of the border in px (for eg- 5px), border_type is used to define whether the border should be dashed or solid line and border_color is the simple border color as usual.

You can also define 'backgroud-position' to set whether the background should be at top, bottom, left or right, center, at % of the total width, top center, bottom center, etc

There is also another property called 'background-attachement', which makes the image scrollable or non-scrollable. Non-scrollable means that the image will be fixed if scrolled.
