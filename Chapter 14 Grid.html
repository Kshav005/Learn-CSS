grid-template-columns

Used to add more columns in a grid. 'grid-template-columns: auto auto auto', will add 3 columns. The number of 'auto' in the property will define the total number of columns. Suppose if you add 5 'auto' in the property then you will be getting 5 columns.

Actually, in place of 'auto', comes a width. CSS asks you about how much width every column should be given. But if you set auto, then it sets the width just same as other columns (divides equally).

grid-gap

A shorthand property, 'grid-gap: row column'. If you enter only one value, then it would set row and column both to that value.

'align-items' and 'justify-content' can also be used in grids. Used to set an alignment in both x and y-axis.

Children Properties

I. grid-column : Used to set a column width according to another column.

To show this as an example, I want to set the column length of the Box1 till Box3. So, it means that the length of the Box1 will so large that it would cover the space of Box1 and Box2 altogether. So the command will go as 'grid-column : 1/3' which means I want the first column to stretch and stop before 3rd box. This means, we will be having 2 columns in the first row and then 3 columns as usual.
