'display : inline' takes up space required by the content. Now you can't change the top and bottom margin/top and bottom padding/width/height. While the left and right padding/margin works good.

This sentence will side by side with the inline box. Therefore, taking up the space it needs.

'display : block' is not a new thing, but it treats the element as block. The 'div' is block by default. Takes full space available and next element comes in the new line. You can change the padding and the margin as usual.

'display : inline-block' is another property in which the item is treated as inline and you can change the width, height of it. If the next element to it has 'display : inline' then both will be side by side.

Well there is one more thing called 'display:none'. It just removes the element. For example, if you look at the code, there is an element but if you check it in the webpage then you won't be able to find such element below.
