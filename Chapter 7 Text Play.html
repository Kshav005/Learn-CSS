Font Family

If you want to change the font of a given text, then we define 'font_family : 'font 1' font2 font3'.

The web browser will select font2 if font1 doesn't work due to some error and same will be font3, if font2 also doesn't work, otherwise, the browser will use it's default font to display the texts. Also, if your font name has a space in between then write it within single quotes (' ') just like I wrote 'font 1'. You can add even more fonts just like this.
