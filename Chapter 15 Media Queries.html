Media Queries? What is a media query? It is not as complicated as it looks but understand it as 'if-else' statements which only activates certain CSS properties when a condition is met.

To begin a media query, we use '@media screen and (window_size : 900px) {property------------------}

So basically, the above code says, when the screen size (height/width) becomes 900px, then perform a property. Let's try doing it with our current webpage.


Now try decreasing the size of the browser window. You will see sudden black when you reach a certain width.
