Transform is pretty popular amongst web devs as it helps in animation of various elements. We can move or scale a particular element just by using properties of CSS. So let's check it out, how everything works.

In this chapter, we will be only learning about the transform property, which helps in 2D or 3D transformation.

Box

So, here's our box today. We will testing out all the properties on it today!

Rotate

We can rotate our box using 'transform : rotate(xturn)' where 'x' is some decimal by which we want that element to turn. 1 is '360 degree'. So we are going to turn this box to 180 degrees by using the value 0.5. Henceforth, the property goes as 'transform : rotate(0.5turn)'

Box

Translate

Translate is nothing but placing an element to a certain position. Use 'transform : translate(x, y)' where x is the place in x-axis while y is the place value in y-axis.

Matrix

Matrix is really simple, it's a shorthand property which goes like 'matrix: a, b, c, d, tx, ty' where a is 'scalex', b is 'skewy', c is 'skewx', d is 'scaley', tx is 'translatex' and ty is 'translatey', containing different values.
