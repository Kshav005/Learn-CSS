In this chapter we are going to talk about the problems faced when a developer uses size or position in px (pixels).


The drawback of px measurement is that it's not relative, which means the size will be same in mobile and PC both. Hence, in PC, it would be normal size but if you look at the webpage from the mobile then it will appear too big for the screen.

In that case, we use some other relative measurements, which fix themselves to either viewing devices or the parent elements.

  • em - unit relative to the parent element's font size.
  • rem - unit relative to the root (browser) font size.
  • vw - unit relative to n% viewport width.
  • vh - unit relative to n% viewport height.

em

If we use this measurement, then the font size becomes *n to the parent element's font size, where n is equal to the number you have mentioned. For eg, '5em' will make the font size *5 of the parent element's size.

rem

This works the same as 'em' but instead of parent element, it uses the default value of the browser. Same usage just like the 'em' measurement.

vw

It will display n% of the viewport width. Viewport is nothing but the size of the screen. So basically, when you type '50vw', the text will be covering 50% of the width of computer/tab/mobile depending on the size.

vh

Works same as 'vw' but instead, it takes the height of the viewport and not the width.

Min/Max-Height

The 'min-height' in CSS is used to set a minimum height for an element. If the height of that element is smaller than minimum height than the element will be set to that minimum height.

Min/Max-Width

Works similar like above, but instead of height, we use width here.

Position Property

There are ways to manipulate the position of an element.

  1. static : The default position, in which top/bottom/left/right makes no effect.

  2. relative : The top/bottom/left/right will work perfectly.

  3. absolute : This element is positioned according to the first non-static element. Also, top/bottom/left/right attribute works as usual.

  4. fixed : Same as absolute but the element is positioned according to browse window

  5. sticky : The element is positioned according to the user's scroll.

Z-Index

It is a simple property which defines the layer of an element. Suppose you want one of your element to be top of another, then make use the 'z-index:1'. By default, the z-index of all elements is 0.
