Flexbox Property

To make things really simple (positions, alignments etc), Flexbox was introduced. It helps in creating navbars and is able to perform many more functions too! It is applied using 'display : flex' in CSS.

There are some terms are important to know, about flexbox.


Property for Flexbox container (parent element, for e.g. - I used 'navbar' as the parent container)

  • flex-direction : Defines the direction in which all the items are laid. Can be row, column, row-reverse, column-reverse. Row is by default.
  • flex-wrap : You can wrap items using this property. Can be wrap, nowrap, wrapreverse etc. It is used to use the space efficiently, when the viewport is small or the user is making the window small.
  • align-items : It defines alignment along the y axis.
  • justify-content : Defines the alignment along the x axis.
  • align-content : Aligns a flex container's lines when there is extra space in y axis. Used alternatively if you don't want to give spaces for flex-wrap, when the user tries to make the window smaller.

Properties for Flexbox items (children elements, for e.g. - I used 'home', 'about', etc as flexbox items)

  • order : Controls the order in which the flexbox items are displayed in the container. By default the order is 0.
  • align-self : Allows to align a certain item, overriding the previous container alignments.
  • flex-grow : Allows the flex item to grow. It increases the size of the item box.
  • flex-shrink : Works same as the 'flex-grow' property but the difference is that it allows to shrink.
