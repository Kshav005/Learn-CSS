Today, we are going to learn about animations! The most coolest feature of CSS and which enhances your webpage to a greater extent. It provides you better, and more controls than transition and transform. We can use '@keyframes' to change the original style to a brand new style!

@keyframes {animation-name} {from {width:15vw;} to{width:25vw;}}

Box

Before continuing further, say hello to our new box which will help us understand animations better!

Now, let's discuss about animation properties.

  • animation-name : The name of the animation, it means you can give any name to your animation. (Works as identifier)
  • animation-duration : Used to set the duration of how long an animation should take place on your webpage.
  • animation-timing-function : Works same as the transition-timing-function.
  • animation-delay : Used to set after how much time, the animation should take place.
  • animation-iteration-count : Used to set the number of times the animation should run of an element.
  • animation-direction : Specifies the direction of the animation
